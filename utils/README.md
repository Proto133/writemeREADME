  ## Installation Instructions 📣
  Clone this repo and run <code>npm i</code> in root directory.
  
  ## Usage 🖱️
  
  -Navigate to root directory of writemeREADME in terminal
  
  -Type ```node index.js```  

  -Answer Prompts  

  -You'll find your generated README in the 'writeme' directory.  
  
  -Enjoy 😎
